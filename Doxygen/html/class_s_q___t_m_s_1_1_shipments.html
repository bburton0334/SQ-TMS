Detailed Description

The purpose of this class is to hold all of the attributes and classes in relation to the shipments carried out by the TMS: Transport Management System.

Details

Shipments class holds the shipmentsID attribue and holds the various classes within it that relate to handeling shipments for the TMS: Transportation Managment System. The attributes include: ShipmentsInfoID. There are three other classes within this main class: Orders, Carriers, Trips.
